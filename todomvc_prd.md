# TodoMVC - Product Requirements Document

## Overview
A simple todo list application where users can manage their daily tasks.

## Features

### 1. Add Todo
- User types a task in the input field
- User presses Enter
- Task appears in the list

### 2. Complete Todo
- User clicks the checkbox next to a task
- Task is marked as completed (strikethrough)

### 3. Delete Todo
- User hovers over a task
- A delete button (X) appears
- User clicks X to remove the task

### 4. Filter Todos
- User can click "All", "Active", or "Completed" to filter the list
