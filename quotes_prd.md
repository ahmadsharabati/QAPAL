# Quotes Site Test

## Feature: Browse quotes across pages

### Scenario 1: View first page of quotes
Navigate to the quotes site homepage at https://quotes.toscrape.com.
Verify there are quotes visible on the page.
Assert the page title contains "Quotes".

### Scenario 2: Navigate to next page
Navigate to the quotes site homepage at https://quotes.toscrape.com.
Click the "Next" navigation link to go to page 2.
Verify the URL has changed to contain "page/2".
Assert that the page still shows quotes content.
