# QAPal Demo PRD

## Feature: User Login
As a user, I want to be able to log into the application using my credentials so that I can access my dashboard.

## Acceptance Criteria
1. The user navigates to the login page at `https://example.com/login`.
2. The user enters their email address into the Email field.
3. The user enters their password into the Password field.
4. The user clicks the "Sign In" button.
5. The user is redirected to the dashboard.
6. The dashboard URL should contain `/dashboard`.
